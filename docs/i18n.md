# Internationalization (i18n)

## Overview

Uses i18next for translations. Write English strings directly in code, then harvest translations.

## Usage

```typescript
import { t } from '@/shared/utils/i18n';

const message = t('feature.name');
```

## Translation Harvesting

Run `pnpm harvest:translations` to extract all translation keys from code.

The script:
1. Scans codebase for `t()` and `i18n.t()` calls
2. Extracts keys and English strings
3. Updates `src/shared/locales/en.json`
4. Creates template files for other languages

## Adding Translations

1. Write code with English strings: `t('feature.setting.label')`
2. Run harvesting script
3. Translation keys appear in `en.json`
4. Add translations to other language files

## Language Detection

Automatically detects browser language. User can override in settings.
