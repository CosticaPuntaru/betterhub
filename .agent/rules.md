# Agentic Rules

These rules must be followed by all agents working on this project.

## Validation Protocol

Before finishing any task, you MUST validate your work in the following order:

1.  **Functional Validation**: Verify that the changes work as expected.
    *   Use the **browser** to verify UI changes and user flows.
    *   Use the **terminal** to verify backend logic or CLI commands.
2.  **TypeScript Validation**: Ensure there are no TypeScript errors.
3.  **Linter Validation**: Ensure there are no linter errors.
