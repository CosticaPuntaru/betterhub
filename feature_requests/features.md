# Feature Requests Index

This folder contains detailed specifications for feasible GitHub feature requests.

## Code Navigation & Review
- [Default "Hide Whitespace"](./hide_whitespace.md) - Automatically hide whitespace in diffs.
- [Sticky File Headers](./sticky_headers.md) - Keep file headers visible while scrolling.
- [NPM/Package Links](./package_links.md) - Clickable links for dependencies in config files.
- [Image Size Display](./image_size.md) - Show dimensions and size for images in diffs.

## Productivity & Workflow
- [Copy File Path Button](./copy_path.md) - One-click button to copy relative file paths.
- ["Viewed" Checkbox Visibility](./viewed_checkbox.md) - Easier access to mark files as viewed.
- [Auto-Expand Resolved Comments](./expand_resolved.md) - Quickly see context of resolved discussions.
- [Prevent Accidental Tab Close](./prevent_close.md) - Warn before closing tabs with unsaved comments.

## Visual Enhancements
- [Reaction Avatars](./reaction_avatars.md) - See who reacted to comments at a glance.
- [Build Status Favicon](./build_favicon.md) - Monitor CI/CD status from the browser tab.
